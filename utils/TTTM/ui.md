## 이미지 뷰어 (ViewerWidget)

- **줌**: Ctrl + 마우스 휠 (0.1x ~ 8.0x)
- **표시 모드**: Original / Threshold / ROI / Mask (콤보박스)
- **ROI 그리기**: "ROI 그리기" 토글 버튼 ON → 이미지 위 드래그
- **Blob 클릭 선택**: ROI 그리기 OFF 상태에서 이미지 클릭 → 해당 blob 자동 선택
- **오버레이**: ROI 사각형(파란색), blob 컨투어(초록/주황)

## 오버레이 비교 다이얼로그 (OverlayDialog)

- 별도 창으로 열림
- 범례: 초록(Image 1) / 빨강(Image 2) / 노랑(겹침)
- 배경: 두 원본 이미지의 평균 그레이스케일
- [Fit] 버튼으로 창 크기에 맞춤

## 로그 뷰어 다이얼로그

- app.log 내용을 실시간 표시
- [새로고침] 버튼으로 최신 로그 반영
