## [2026-06-06] 알려진 제한사항 (v1.0~v1.3, 이전 작업 환경에서 이관)
- 증상: 16-bit 단채널 RAW(little-endian) 이미지 전용이며 cv2.imread 사용 불가. 한 번에 이미지 1쌍만 비교 가능. 이미지당 blob 1개만 선택 가능. ROI는 사각형만 지원. 처리 중 취소 버튼이 없어 대용량 이미지에서 UI 블로킹 가능.
- 원인: 초기 설계 범위를 16-bit RAW 단일 채널 비교 용도로 한정.
- 해결: 미해결 (설계상 제약). 별도 요청 시 개선 검토.
- 상태: 미해결
